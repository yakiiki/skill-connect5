 🎯 技能五子棋 (Skill Gomoku)

一款融合了中国成语技能的策略五子棋游戏，基于 GitHub Pages 自动托管。

## 🎮 游戏特色

### 核心玩法
- **Ban/Pick 环节**：对局前互相禁用3个技能，自选3个技能
- **技能系统**：30个基于中国成语的独特技能
- **策略对战**：传统五子棋 + 技能策略 = 全新体验

### 技能类型
- **攻击型**：声东击西、釜底抽薪、天下无双等
- **防御型**：围魏救赵、金蝉脱壳、完璧归赵等
- **控制型**：关门捉贼、偷梁换柱、调虎离山等
- **特殊型**：草船借箭、浑水摸鱼、八面玲珑等

### 等级系统
- 最高等级：**10级**
- 升级所需经验：等级 × 100
- 每完成一局：+10经验
- 获胜：额外+20经验

## 🎯 游戏规则

### 游戏流程
1. **Ban阶段**：双方各选择3个技能禁用
2. **Pick阶段**：从可用技能中选择3个
3. **对战阶段**：黑方先行，轮流落子

### 技能使用
- 每局有**3次**使用技能的机会
- 每个技能每局只能使用**1次**
- 技能可在自己回合使用

### 胜利条件
率先在横、竖、斜方向连成5子即获胜。
